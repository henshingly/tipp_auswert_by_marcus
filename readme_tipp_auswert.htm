Änderungen
Folgende Änderungen sind durchzuführen:

1.) Heruntergeladene Datei (tipp_auswert.rar) entpacken

2.) Datei auf Server kopieren
\auswert.php -> ins LMO-Root

>> www.bcerlbach.de mal besuchen ;) <<

-----[ SAVE/CLOSE ALL FILES ]------------------------------------------

3.) Auswertskript in den LMO einbinden 

# 
#-----[ öffne die Datei ] ------------------------------------------
# 

\lmo4\addon\tipp\lmo-tipppad.php


# 
#-----[ FINDE Zeile 56 ]------------------------------------------ 
# 

  if($tipp_gesamt==1 && ($u>2 || $tipp_nurgesamt==1 && $u==2)){
    $auswertfile=PATH_TO_ADDONDIR."/tipp/".$tipp_dirtipp."auswert/gesamt.aus";?>
        <li>
          <a href="<?=$addw."&amp;all=1"; ?>"><strong><?=$text['tipp'][25];?></strong></a><?if(file_exists($auswertfile))
{echo "<br><small>".$text['tipp'][83].": ".date("d.m.Y H:i",filemtime($auswertfile))."</small>";} ?>
        </li>


# 
#-----[ einfügen danach ]------------------------------------------ 
# 
	<li>
          <a href="auswert.php" target="_self"><strong>Individuelle Auswertung</strong></a>
        </li>


#
#-----[ Hinweis ]------------------------------------------
#        
    Target kann jeder selber seinen Wünschen anpassen:
		- target="_self"	oder
		- target="_blank"	oder
		- target="_parant"	oder
		- target="_top"


Info: Auf Wunsch kann man das Skript nur für eingeloggt User zugänglich machen: 

# 
#-----[ öffne die Datei ] ------------------------------------------
# 

\lmo4\auswert.php


# 
#-----[ FINDE Zeile 97 ]------------------------------------------ 
# 

/*if ($_SESSION["lmotipperok"] != 5) {
		die("Sorry, aber der Zugang ist nur für eingeloggte User möglich!");
}*/

# 
#-----[ DANACH einfügen ]------------------------------------------ 
# 

if ($_SESSION["lmotipperok"] != 5) {
		die("Sorry, aber der Zugang ist nur für eingeloggte User möglich!");
}


--> die Individuellen Skripteinstellungen kann man in den Zeilen von 34 bis 89 in der auswert.php ändern
